## 1. Konfigürasyon Özeti

| Özellik       | Değer                                         |
| ------------- | --------------------------------------------- |
| **Dosya**     | `.storybook/main.ts`                          |
| **Framework** | Vue 3 + Vite                                  |
| **Stories**   | `*.mdx`, `*.stories.@(js\|jsx\|mjs\|ts\|tsx)` |

---

## 2. Stories Deseni

```ts
stories: [
  '../src/**/*.mdx',
  '../src/**/*.stories.@(js|jsx|mjs|ts|tsx)',
]
```

| Pattern                                        | Açıklama                    |
| ---------------------------------------------- | --------------------------- |
| `../src/**/*.mdx`                              | MDX dokümantasyon dosyaları |
| `../src/**/*.stories.@(js\|jsx\|mjs\|ts\|tsx)` | Hikâye dosyaları            |

---

## 3. Addon'lar

| Addon                      | Açıklama                        |
| -------------------------- | ------------------------------- |
| `@storybook/addon-a11y`    | Erişilebilirlik (a11y) kontrolü |
| `@storybook/addon-docs`    | Otomatik dokümantasyon          |
| `@storybook/addon-designs` | Tasarım araçları entegrasyonu   |

> **Not:** Chromatic görsel regression testleri `build:chromatic` komutu ile CLI üzerinden çalışır; addon olarak eklenmez.

---

## 4. Framework

```ts
framework: '@storybook/vue3-vite'
```

Vue 3 ve Vite ile Storybook çalışır. Vite build pipeline'ı kullanılır.

---

## 5. İlgili Dosyalar

| Dosya                   | Açıklama                                         |
| ----------------------- | ------------------------------------------------ |
| `.storybook/main.ts`    | Ana Storybook config                             |
| `.storybook/preview.ts` | Global decorator, parametreler                   |
| `.storybook/routes.ts`  | Vue Router rotaları (storybook-vue3-router için) |

## 6. Komutlar

| Komut                     | Açıklama                                          |
| ------------------------- | ------------------------------------------------- |
| `npm run storybook`       | Geliştirme sunucusu (port 6006)                   |
| `npm run build:storybook` | Statik Storybook build                            |
| `npm run build:chromatic` | Storybook build + Chromatic'e yükle (görsel test) |

## 7. Hikâye Dosyası Örneği

Projede mevcut örnek: `src/views/Dashboard/Ecommerce.stories.ts`

```ts
// src/views/Dashboard/Ecommerce.stories.ts
import type {
  Meta,
  StoryObj,
} from '@storybook/vue3'
import EcommerceView from './EcommerceView.vue'

const meta: Meta<typeof EcommerceView> = {
  component: EcommerceView,
  tags: ['autodocs'],
  parameters: {
    layout: 'fullscreen',
  },
}
export default meta

type Story = StoryObj<typeof EcommerceView>

export const Default: Story = {}
```

Mevcut story dosyaları: `src/views/Dashboard/Ecommerce.stories.ts`, `src/stories/Welcome.mdx`
